# E-commerce-Sales-Customer-Insights-Dashboard
End-to-end data analytics project analyzing e-commerce customer behavior using Python, SQL, and Power BI. Includes data cleaning, EDA, SQL analysis, and interactive dashboards to uncover insights on sales, customers, and product performance.


E-commerce Sales & Customer Insights Dashboard
Project Overview

This is an end-to-end data analytics project focused on analyzing e-commerce customer behavior and sales performance. The project covers the complete data analytics workflow including data cleaning, exploratory data analysis (EDA), SQL-based analysis, and interactive dashboard creation using Power BI.

Objective

To analyze customer purchase behavior, identify key sales trends, and generate actionable insights to support business decision-making.

Tech Stack
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (MySQL / phpMyAdmin)
Power BI
Excel (for initial exploration)
Project Workflow
1. Data Collection
Used customer shopping behavior dataset containing purchase details, demographics, and transaction information
2. Data Cleaning & Preprocessing
Handled missing values in age and categorical fields
Removed duplicate records
Standardized categorical values (gender, category, etc.)
Converted data types (dates, numeric columns)
3. Exploratory Data Analysis (EDA)
Analyzed sales distribution across categories and regions
Identified customer behavior patterns
Studied impact of discounts on purchase behavior
Explored age group and gender-based trends
4. SQL Analysis
Wrote queries to extract insights such as:
Total revenue and orders
Top-performing categories
Customer segmentation
Discount usage patterns
5. Power BI Dashboard

Created interactive dashboards with:

KPI Cards (Revenue, Orders, AOV, Discount Rate, Repeat Customers)
Sales performance analysis
Customer segmentation insights
Category-wise revenue distribution
Seasonal trends and discount impact
Key Insights
Clothing category contributes the highest revenue
Seasonal trends influence sales performance
Discounts increase order volume but reduce average order value
A small percentage of customers contribute major revenue
Subscribers and repeat customers drive higher engagement
Business Recommendations
Focus marketing on high-performing categories
Optimize discount strategy to maintain profitability
Target repeat customers with loyalty programs
Convert non-subscribers into subscribers
Personalize marketing based on customer segments
